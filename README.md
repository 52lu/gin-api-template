# gin-import-template
Gin框架学习使用,实践常用包在Gin框架中的集成和使用

# 目录介绍
```
├── api // 接口
├── config // 配置
├── core // 核心代码
├── global // 全局变量和常量
├── initialize // 初始化相关
├── logs // 日志目录
├── main.go // 启动文件
├── model // 实体
├── router // 路由
    └── middleware //中间件
├── test // 单元测试目录
└── utils // 工具包
```
